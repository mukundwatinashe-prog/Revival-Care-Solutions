# Logos Directory

Please add the following logo files to this directory:

1. **care-inspectorate-logo.png** - Care Inspectorate logo
   - Recommended size: 280x120px or larger
   - Format: PNG with transparent background preferred
   - This logo will be displayed in the footer

2. **falkirk-council-logo.png** - Falkirk Council logo
   - Recommended size: 280x120px or larger
   - Format: PNG with transparent background preferred
   - This logo will be displayed in the footer

## Notes

- The logos will be displayed in white containers in the footer
- They will automatically scale to fit the container (140x60px display size)
- If logos are not available, the footer will show broken images until you add them
- You can use any format (PNG, JPG, SVG) but PNG with transparency works best
